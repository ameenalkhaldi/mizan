## ما

اسأل هذه الأسئلة بالترتيب:

```
هل جاءت بعد "إنّ/أنّ/كأنّ/لكنّ/ليت/لعلّ/رُبّ/حيث/بعد"؟
├── نعم → ما الزائدة (الكافة): كفّت العامل عن العمل
│   إنّما العلمُ نورٌ (كافة لإنّ)
│   ربّما يأتي (كافة لربّ)
│   حيثما تذهبْ أذهبْ (كافة لحيث)
│
└── لا ←
    هل جاءت قبل "أفعلَ" على وزن التعجب + منصوب؟
    ├── نعم → ما التعجبية: مبتدأ، في محل رفع
    │   ما أجملَ السماءَ!
    │
    └── لا ←
        هل جاءت في موقع يليها فعل وتدل على شرط (+ جزم)؟
        ├── نعم → ما الشرطية: مبنية، إعرابها حسب موقعها
        │   ما تفعلْ تُجْزَ به
        │
        └── لا ←
            هل جاءت في بداية استفهام أو يُفهم منها السؤال؟
            ├── نعم → ما الاستفهامية: مبنية، حسب موقعها
            │   ما هذا؟ / ماذا فعلتَ؟
            │   (ملاحظة: تُحذف ألفها إذا دخل عليها حرف جر: لِمَ؟ عمَّ؟ بِمَ؟ فيمَ؟)
            │
            └── لا ←
                هل جاءت قبل فعل ماضٍ/مضارع ويصح تأويلها بمصدر؟
                ├── نعم → ما المصدرية
                │   أعجبني ما صنعتَ (= صنعُك)
                │   (إذا كانت مع "دام" → مصدرية ظرفية: ما دمتُ حياً)
                │
                └── لا ←
                    هل جاءت قبل جملة اسمية وتنفي المعنى؟
                    ├── نعم → هل استوفت شروط الحجازية (لم تُقترن بإنْ، لم يتقدم الخبر، لم يُنتقض بإلا)؟
                    │   ├── نعم → ما الحجازية (عاملة عمل ليس)
                    │   │   ما هذا بشراً
                    │   └── لا → ما النافية المهملة (تميمية)
                    │       ما زيدٌ قائمٌ
                    │
                    └── لا ←
                        هل جاءت قبل فعل وتنفي معناه؟
                        ├── نعم → ما النافية (لا عمل لها مع الأفعال)
                        │   ما ذهبَ زيدٌ / ما يذهبُ زيدٌ
                        │
                        └── لا ←
                            هل يصح وضع "الذي/التي" مكانها + لها صلة وعائد؟
                            ├── نعم → ما الموصولة: مبنية، حسب موقعها
                            │   أعجبني ما عندك (= الذي عندك)
                            │
                            └── لا → ما الزائدة (في غير النواسخ)
                                مثل: إذا ما جئتَ (ما: زائدة بعد إذا)
```

## لا

```
هل جاءت قبل فعل مضارع + طلب (أمر/نهي)؟
├── نعم → لا الناهية (جازمة): تجزم المضارع
│   لا تكتبْ / لا تذهبْ
│
└── لا ←
    هل جاءت قبل اسم نكرة + يُقصد نفي الجنس كله؟
    ├── نعم → هل استوفت شروطها (اسمها نكرة، لم تُفصل عنه، لم تُسبق بحرف جر)؟
    │   ├── نعم → لا النافية للجنس (تعمل عمل إنّ)
    │   │   لا رجلَ في الدارِ
    │   └── لا → لا النافية المهملة (تُكرر أو تُهمل)
    │       لا في الدارِ رجلٌ ولا امرأةٌ
    │
    └── لا ←
        هل جاءت قبل اسم مرفوع + خبر ويصح معنى "ليس"؟
        ├── نعم → لا العاملة عمل ليس (نادرة)
        │   لا رجلٌ أفضلَ منك (= ليس رجلٌ أفضلَ)
        │
        └── لا ←
            هل جاءت بين اسمين معطوفين (نفي الثاني)؟
            ├── نعم → لا العاطفة: حرف عطف
            │   جاء زيدٌ لا عمرٌو
            │
            └── لا ←
                هل جاءت قبل فعل مضارع/ماضٍ بلا طلب؟
                ├── نعم → لا النافية (لا عمل لها)
                │   لا يذهبُ زيدٌ / لا ذهبَ زيدٌ (نادر مع الماضي)
                │
                └── لا → لا الزائدة (للتوكيد)
                    {مَا مَنَعَكَ أَلَّا تَسْجُدَ} (لا: زائدة)
```

## أنْ وإنْ

### أنْ (بفتح الهمزة)

```
هل يليها فعل مضارع (منصوب)؟
├── نعم → أنْ المصدرية الناصبة
│   أريدُ أنْ أذهبَ (= الذهابَ)
│
└── لا ←
    هل يليها فعل (ماضٍ أو أمر أو مضارع غير منصوب) وتُفسّر ما قبلها؟
    ├── نعم → أنْ التفسيرية (حرف لا محل له)
    │   أشرتُ إليه أنِ اذهبْ
    │   (شروطها: مسبوقة بما فيه معنى القول دون حروفه)
    │
    └── لا ←
        هل يليها جملة (اسمية أو فعلية) ويصح تقدير "أنّ" المشددة + ضمير شأن محذوف؟
        ├── نعم → أنْ المخففة من أنّ
        │   علمتُ أنْ سيكونُ الأمرُ سهلاً (أصلها: أنّه)
        │   (علامتها: يأتي بعدها "قد/سوف/السين/لا/لم" أو فعل جامد)
        │
        └── لا → أنْ الزائدة
            لمّا أنْ جاءَ البشيرُ (أنْ: زائدة)
```

### إنْ (بكسر الهمزة)

```
هل يليها فعل مضارع مجزوم (أو ماضٍ بمعنى الشرط)؟
├── نعم → إنْ الشرطية الجازمة
│   إنْ تدرسْ تنجحْ
│
└── لا ←
    هل تنفي الجملة بعدها ويصح معنى "ما/ليس"؟
    ├── نعم → هل بعدها اسم + خبر منصوب؟
    │   ├── نعم → إنْ النافية العاملة عمل ليس (نادرة)
    │   └── لا → إنْ النافية المهملة (الأكثر)
    │       إنْ زيدٌ إلا كريمٌ (= ما زيدٌ إلا كريمٌ)
    │       (غالباً تقترن بـ "إلا" أو اللام الفارقة)
    │
    └── لا ←
        هل يليها جملة + لام الابتداء (اللام الفارقة) في الخبر؟
        ├── نعم → إنْ المخففة من إنّ (التوكيدية)
        │   إنْ زيدٌ لَكريمٌ (= إنّ زيداً لَكريمٌ)
        │
        └── لا → إنْ الزائدة
            ما إنْ زيدٌ قائمٌ (إنْ: زائدة بعد ما)
```

## الواو

```
هل الواو قبل اسم مجرور + في سياق قسم؟
├── نعم → واو القسم (حرف جر)
│   واللهِ لأفعلنّ
│
└── لا ←
    هل الواو قبل جملة اسمية/حالية + بعد جملة فعلية + فيها ضمير يعود على فاعل؟
    ├── نعم → واو الحال (حرف لا محل له)
    │   جاء زيدٌ وهو يضحكُ
    │   (علامتها: يصح تقدير "في حال" مكانها)
    │
    └── لا ←
        هل الواو قبل اسم منصوب + بعد فعل + بمعنى "مع" + لا يصح العطف؟
        ├── نعم → واو المعية (حرف)
        │   سرتُ والنيلَ
        │   (المعيار: هل ما بعدها شارك في الفعل؟ لا → معية)
        │
        └── لا ←
            هل الواو بين جملتين مستقلتين لا رابط بينهما (موضوع جديد)؟
            ├── نعم → واو الاستئناف (حرف لا محل له)
            │   نجحَ زيدٌ. ورسبَ عمرٌو.
            │
            └── لا ←
                هل الواو في أول كلام + قبل اسم نكرة مجرور بربّ المحذوفة؟
                ├── نعم → واو ربّ (حرف جر = ربّ المحذوفة)
                │   وليلٍ كموجِ البحرِ (= ربّ ليلٍ)
                │
                └── لا → واو العطف (الافتراضي الأكثر شيوعاً)
                    جاء زيدٌ وعمرٌو
```

## الفاء

```
هل الفاء في جواب شرط (بعد أداة شرط جازمة + فعل الشرط)؟
├── نعم → فاء الجزاء / فاء جواب الشرط (رابطة)
│   إنْ تدرسْ فأنتَ ناجحٌ
│   (الجملة بعدها في محل جزم)
│
└── لا ←
    هل الفاء بعد نفي/طلب + قبل فعل مضارع منصوب + بمعنى السببية؟
    ├── نعم → فاء السببية (ينتصب المضارع بعدها بأنْ مضمرة)
    │   ادرسْ فتنجحَ / لا تكسلْ فتخسرَ
    │
    └── لا ←
        هل الفاء بين مفردين أو جملتين + بمعنى الترتيب والتعقيب؟
        ├── نعم → فاء العطف (ترتيب وتعقيب)
        │   جاء زيدٌ فعمرٌو (= بعده مباشرة)
        │
        └── لا ←
            هل الفاء تقع بعد كلام محذوف يُفهم من السياق؟
            ├── نعم → الفاء الفصيحة (تفصح عن شرط محذوف)
            │   {فَقُلْنَا اضْرِبْ بِعَصَاكَ الْحَجَرَ فَانْفَجَرَتْ} (= فضربَ فانفجرت)
            │
            └── لا → فاء الاستئناف (حرف لا محل له)
                نجحَ زيدٌ، فهل حضرَ الحفلَ؟
```

## مِن / مَن

### مِن (بكسر الميم — حرف)

```
هل جاءت بعد نفي/استفهام/نهي + قبل نكرة + يصح حذفها دون تغيير المعنى الأساسي؟
├── نعم → مِن الزائدة: ما بعدها مجرور لفظاً لكن في محل آخر
│   هل مِن أحدٍ هنا؟ (أحدٍ: مجرور لفظاً في محل رفع مبتدأ)
│   ما جاء مِن أحدٍ (أحدٍ: مجرور لفظاً في محل رفع فاعل)
│
└── لا → مِن الأصلية: حرف جر، يفيد أحد المعاني
    ابتداء الغاية: خرجتُ مِن البيتِ
    التبعيض: أكلتُ مِن الطعامِ
    البيان: {فَاجْتَنِبُوا الرِّجْسَ مِنَ الْأَوْثَانِ}
    التعليل: مِن حسدِهم
```

### مَن (بفتح الميم — اسم)

```
هل في سياق استفهام؟
├── نعم → مَن الاستفهامية: مبنية، حسب موقعها
│   مَن جاء؟ (مبتدأ) / مَن رأيتَ؟ (مفعول به)
│
└── لا ←
    هل يليها فعل مجزوم (فعل شرط)؟
    ├── نعم → مَن الشرطية: مبنية، حسب موقعها
    │   مَن يدرسْ ينجحْ
    │
    └── لا → مَن الموصولة (= الذي): مبنية، حسب موقعها
        جاء مَن نجحَ (= الذي نجحَ)
```

## الباء

```
هل الباء في سياق قسم (باللهِ)؟
├── نعم → باء القسم: حرف جر وقسم
│
└── لا ←
    هل الباء بعد "ليس/ما الحجازية" + في الخبر، أو بعد نفي + في الفاعل/المفعول، أو قبل "كافٍ/كفى"؟
    ├── نعم → الباء الزائدة: ما بعدها مجرور لفظاً في محل آخر
    │   ليس زيدٌ بكسولٍ (كسولٍ: في محل نصب خبر ليس)
    │   كفى باللهِ شهيداً (اللهِ: في محل رفع فاعل)
    │   ما زيدٌ بقائمٍ (قائمٍ: في محل رفع خبر أو نصب خبر ما)
    │
    └── لا → الباء الأصلية: حرف جر (إلصاق/استعانة/سببية/مصاحبة/ظرفية/بدل/قَسَم)
        كتبتُ بالقلمِ (استعانة)
        فرحتُ بنجاحِك (سببية)
        مررتُ بزيدٍ (إلصاق)
```

## اللام

```
هل اللام مكسورة + قبل فعل مضارع مجزوم + في سياق طلب؟
├── نعم → لام الأمر (جازمة)
│   لِيكتبْ زيدٌ
│
└── لا ←
    هل اللام قبل فعل مضارع منصوب (بأنْ مضمرة)؟
    ├── نعم → هل سبقتها "ما كان/لم يكن"؟
    │   ├── نعم → لام الجحود (+ نصب بأنْ مضمرة)
    │   │   ما كان اللهُ لِيُضيعَ إيمانَكم
    │   └── لا → لام التعليل (+ نصب بأنْ مضمرة)
    │       جئتُ لأتعلمَ
    │
    └── لا ←
        هل اللام على اسم في خبر إنّ أو أول جملة مؤكدة؟
        ├── نعم → هل هي مع إنّ؟
        │   ├── نعم → اللام المزحلقة (لام الابتداء زُحلقت إلى الخبر)
        │   │   إنّ العلمَ لَنافعٌ
        │   └── لا → لام الابتداء (للتوكيد، تدخل على المبتدأ أو الخبر)
        │       لَزيدٌ كريمٌ
        │
        └── لا ←
            هل اللام في جواب قسم + قبل "قد" أو فعل مضارع + نون التوكيد؟
            ├── نعم → لام القسم (الموطّئة أو الواقعة في الجواب)
            │   واللهِ لأفعلنّ / لقد فعلتُ
            │
            └── لا ←
                هل اللام في "لئن"؟
                ├── نعم → اللام الموطِّئة للقسم: تدل على أن الجواب الآتي للقسم لا للشرط
                │   لئنْ درستَ لتنجحنّ
                │
                └── لا → لام الجر (ملك/اختصاص/تعليل/شبه ملك)
                    الكتابُ لزيدٍ (ملك)
                    يا لَلعجبِ (تعجب)
```

## أل

```
هل أل داخلة على اسم فاعل/مفعول يعمل عمل فعله (= بمعنى الذي يفعل/يُفعل)؟
├── نعم → أل الموصولة (= الذي/التي)
│   جاء الضاربُ زيداً (= الذي يضربُ زيداً)
│   المقروءُ كتابُهُ (= الذي يُقرأ كتابُه)
│
└── لا ←
    هل الاسم سبق ذكره أو هو معهود ذهنياً؟
    ├── نعم → أل العهدية
    │   ذهبتُ إلى المدرسةِ (العهد الذهني — مدرسة معيّنة)
    │   جاء رجلٌ، فقال الرجلُ... (العهد الذكري — سبق ذكره)
    │
    └── لا ←
        هل يصح تقدير "كل" مكان أل (= جميع أفراد الجنس)؟
        ├── نعم → أل الاستغراقية
        │   الإنسانُ ضعيفٌ (= كل إنسان)
        │
        └── لا → أل الجنسية (لبيان الحقيقة/الماهية)
            الذهبُ معدنٌ نفيسٌ (= جنس الذهب)
```

## أنّ وإنّ (المشددتان)

```
هل الهمزة في بداية الكلام المطلق أو بعد "قال" الحقيقي أو بعد "ألا" الاستفتاحية أو بعد "حيث"؟
├── نعم → إنّ (بكسر الهمزة)
│   إنّ العلمَ نورٌ / قال إنّ العلمَ نافعٌ / ألا إنّ...
│
└── لا ←
    هل يمكن تأويلها مع معموليها بمصدر يحل محلها في الجملة (فاعل/مفعول/مبتدأ/مجرور)؟
    ├── نعم → أنّ (بفتح الهمزة) — مصدر مؤول
    │   سرّني أنّك ناجحٌ (= سرّني نجاحُك → فاعل)
    │   علمتُ أنّك قادمٌ (= علمتُ قدومَك → مفعول به)
    │
    └── لا ←
        هل هي بعد فعل قول غير حقيقي (= في الحكاية والنقل)؟
        ├── نعم → يجوز الوجهان: إنّ (حكاية القول) أو أنّ (مصدر مؤول)
        │
        └── لا → راجع القاعدة: إذا صحّ مكانها "المصدر" → أنّ بالفتح، وإلا → إنّ بالكسر
```

## جعل

```
هل "جعل" بمعنى "بدأ/شرع" + يليها فعل مضارع بلا "أنْ"؟
├── نعم → جعل من أفعال الشروع (تعمل عمل كاد)
│   جعل الولدُ يبكي (يبكي: خبرها)
│
└── لا ←
    هل "جعل" بمعنى "صيّر/حوّل" + يليها مفعولان (اسم + وصف)؟
    ├── نعم → جعل من أفعال التحويل (تعمل عمل ظنّ — تنصب مفعولين)
    │   جعل اللهُ الأرضَ ممهّدةً
    │
    └── لا ←
        هل "جعل" بمعنى "خلق/أوجد" + مفعول واحد؟
        ├── نعم → جعل بمعنى خلق (فعل متعدٍّ لمفعول واحد)
        │   {وَجَعَلَ الظُّلُمَاتِ وَالنُّورَ}
        │
        └── لا → تحتاج سياقاً أوسع لتحديد المعنى
```

## تحديد تعدّي الفعل (لازم / متعدٍّ)

### القاعدة
- **المتعدي**: ينصب مفعولاً به بنفسه (فتحتُ البابَ)
- **اللازم**: لا ينصب مفعولاً به بنفسه (ذهبَ زيدٌ)

### اختبار سريع
**هل يصح أن تسأل "ماذا فعلَ؟" ويكون الجواب اسماً منصوباً مباشرة بلا حرف جر؟**
- كتبَ ← ماذا كتبَ؟ الرسالةَ ✓ → متعدٍّ
- ذهبَ ← ماذا ذهبَ؟ ✗ (لا يصح) → لازم
- أعطى ← ماذا أعطى؟ زيداً كتاباً ✓ → متعدٍّ لمفعولين

### أوزان الأفعال ودلالتها على التعدي

#### الثلاثي المجرد — يختلف بحسب الباب (سماعي في الغالب)
- **الأغلب متعدٍّ**: فَعَلَ يَفعُلُ (نَصَرَ)، فَعَلَ يَفعِلُ (ضَرَبَ)
- **الأغلب لازم**: فَعُلَ يَفعُلُ (كَرُمَ)، فَعِلَ يَفعَلُ (فَرِحَ — إذا دلّ على صفة/انفعال)
- **لكن هناك استثناءات كثيرة — المرجع هو المعنى**

#### المزيد — أنماط منتظمة

| الوزن | الحكم الغالب | مثال |
|-------|-------------|------|
| أفعلَ (بهمزة التعدية) | **متعدٍّ** (يُعدّي اللازم) | أجلسَ (من جلسَ اللازم) |
| فعَّلَ (بالتضعيف) | **متعدٍّ** (يُعدّي اللازم أو يزيد مفعولاً) | فرَّحَ (من فرِحَ)، علَّمَ (من علِمَ — زاد مفعولاً) |
| فاعَلَ | **متعدٍّ** غالباً | كاتَبَ، قاتَلَ |
| انفعلَ | **لازم** دائماً | انكسرَ، انطلقَ |
| افتعلَ | **يختلف** (متعدٍّ أو لازم) | اكتسبَ (متعدٍّ) / اجتمعَ (لازم) |
| تفعَّلَ | **لازم** غالباً (مطاوع فعَّلَ) | تعلَّمَ، تكسَّرَ |
| تفاعلَ | **لازم** غالباً (مطاوع فاعلَ) | تقاتلَ، تبادلَ |
| استفعلَ | **متعدٍّ** غالباً | استخرجَ، استعملَ |
| افعلَّ (بتشديد اللام) | **لازم** دائماً (لون/عيب) | احمرَّ، اعوجَّ |

### تعدّي الفعل بالحرف
الفعل اللازم يصل إلى المفعول بحرف جر:
- ذهبَ **إلى** المدرسةِ (لازم + حرف جر)
- مرَّ **بـ**زيدٍ

### كيف تتعامل مع فعل لا تعرف تعديه؟
1. جرّب اختبار "ماذا فعلَ؟"
2. انظر إلى الوزن (أنماط أعلاه)
3. انظر إلى السياق: هل يوجد اسم منصوب بعده بلا حرف جر؟
4. إذا كان الفعل من أفعال الحركة/الحالة/الصفة → غالباً لازم
5. إذا كان الفعل يدل على تأثير في مفعول خارجي → غالباً متعدٍّ

## فكّ التباس النص غير المشكّل

### المنهجية المتسلسلة

عند غياب التشكيل، اتبع هذه الخطوات:

#### الخطوة 1: حدد أقسام الكلام أولاً (اسم/فعل/حرف)

**علامات الفعل:**
- قبول تاء التأنيث الساكنة (كتبَتْ) → ماضٍ
- قبول "قد" (قد كتبَ / قد يكتبُ) → ماضٍ أو مضارع
- حروف المضارعة (أ/ن/ي/ت) في أول الكلمة + وزن مضارع → مضارع
- قبول "لم" (لم يكتبْ) → مضارع
- قبول ياء المخاطبة (اكتبي) → أمر

**علامات الاسم:**
- أل التعريف (الكتاب)
- التنوين (كتابٌ)
- دخول حرف جر عليه (في المدرسة)
- الإضافة (كتابُ المعلمِ)
- النداء (يا زيدُ)
- الإسناد إليه (العلمُ نورٌ — أُسند إليه خبر)

**علامات الحرف:**
- لا يقبل أيّاً من علامات الاسم أو الفعل

#### الخطوة 2: حدد هل الفعل مبني للمعلوم أو للمجهول

**قرائن المبني للمجهول:**
- لا فاعل ظاهر بعد الفعل + الاسم بعده هو المتلقي (كُتبت الرسالة)
- السياق يقتضي المبني للمجهول (الفاعل مجهول أو غير مهم)
- وجود "نائب فاعل" واضح (مَن لا يصح أن يكون فاعلاً)

**القرينة الحاسمة:** هل الاسم المرفوع بعد الفعل هو الذي فعل أو الذي فُعل به؟
- "كتب الطالب الرسالة" → الطالب فعل → معلوم
- "كتبت الرسالة" (بلا فاعل مذكور) → الرسالة فُعلت → قد يكون مجهولاً

#### الخطوة 3: حدد بنية الجملة

**الجملة الاسمية:**
- تبدأ باسم (بما فيه الضمائر)
- أو بحرف ناسخ (إنّ وأخواتها) + اسم

**الجملة الفعلية:**
- تبدأ بفعل

**القرينة:** الكلمة الأولى — إذا كانت فعلاً → فعلية، إذا كانت اسماً → اسمية

#### الخطوة 4: في حال الالتباس الحقيقي

إذا كانت القراءتان ممكنتين فعلاً (لا يحسمها السياق):
1. اذكر القراءة الأرجح وأعربها كاملاً
2. أشِر إلى القراءة البديلة تحت "ملاحظات" مع الفرق في الإعراب

**أمثلة على التباس حقيقي:**
- "علم المعلم الطالب الدرس":
  - عَلَّمَ المعلمُ الطالبَ الدرسَ (فعلية: المعلم فاعل)
  - عِلمُ المعلمِ الطالبَ الدرسَ ... (اسمية: عِلم مبتدأ — يحتاج خبراً)
  - السياق يرجّح الأولى (جملة تامة)

- "ضرب موسى عيسى":
  - ضربَ موسى عيسى: الفاعل موسى (المقدّم) — القاعدة: إذا التبس الفاعل والمفعول، الفاعل هو المتقدم رتبةً

## تحديد حدود الجمل والنطاق

### قاعدة نطاق العامل

كل عامل يعمل فيما يليه حتى يأتي ما يقطعه (عامل آخر أو نهاية الجملة).

### علامات بداية جملة جديدة (مستأنفة)

| العلامة | مثال |
|---------|------|
| حرف استئناف (الفاء/الواو الاستئنافية) + تغيّر الموضوع | نجح زيدٌ. وذهبَ عمرٌو |
| "أمّا ... فـ" | أمّا زيدٌ فذاهبٌ |
| حرف جواب (نعم/بلى/أجل) | نعم، ذهبَ |
| أداة شرط جديدة | إذا/إنْ/مَن... |
| فعل أمر مستقل | اذهبْ. اكتبْ |

### تحديد نطاق النواسخ (كان/إنّ/كاد...)

**اسم الناسخ**: أول اسم مرفوع (لكان) أو منصوب (لإنّ) بعد الناسخ
**خبر الناسخ**: يمتد حتى اكتمال المعنى:
- إذا كان مفرداً: الكلمة التالية المناسبة
- إذا كان جملة: حتى نهاية الجملة الفرعية
- إذا كان شبه جملة: الجار والمجرور أو الظرف

### تحديد مرجع الضمير المستتر

**القاعدة:** الضمير المستتر يعود على أقرب مذكور مناسب (في الجنس والعدد):
- زيدٌ يكتبُ (يكتبُ: فاعله "هو" = زيد)
- هندٌ تكتبُ (تكتبُ: فاعله "هي" = هند)
- إذا تعدد المرجع المحتمل: يعود على الأقرب ما لم يمنع المعنى

### تحديد ما يتبع الجملة وما يبدأ جملة جديدة

**التابع يلي متبوعه مباشرة (أو بفاصل قصير):**
- النعت يلي المنعوت
- المعطوف يلي المعطوف عليه
- التوكيد يلي المؤكد
- البدل يلي المبدل منه

**إذا جاء اسم بعد جملة تامة ولا يصح أن يكون تابعاً لشيء فيها → بداية جملة جديدة**

### نطاق الاستثناء بـ "إلا"
"إلا" + المستثنى يتبعان المستثنى منه. إذا لم يوجد مستثنى منه → استثناء مفرغ (ما بعد "إلا" حسب موقعه).

### تداخل الجمل — الترتيب
عند تداخل عدة جمل (صلة، حال، خبر، نعت)، حلّل من الداخل إلى الخارج:
1. حدد أصغر جملة مضمّنة أولاً
2. أعربها كاملاً
3. حدد محلّها (صلة/حال/خبر/نعت/مضاف إليه)
4. انتقل إلى الجملة الأكبر المحيطة بها
5. كرر حتى تصل إلى الجملة الرئيسية
